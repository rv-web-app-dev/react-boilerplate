# Documentation
This project is a boiler plate react app ready to be used for any react app projects.

## How to install?
`npm install`

## How to Start?
`npm start`

## Where does it start?
[https://localhost:9000/](https://localhost:9000/)


##Modules used:
- Babel7
- Webpack4
- React16
